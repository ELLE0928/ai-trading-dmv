AI Trading DMV

A sandbox road test for AI trading agents before they receive trading permissions.

Simulation only External agent API No real orders
Purpose

Decides whether an AI trader can enter paper trading or higher permissions

AI Trading DMV is a pre-live sandbox exam. It checks whether a trading agent can act safely, explain its behavior, and stay inside its license before touching capital.

Suspended L0 Read-only L1 Paper L2 Assisted Spot
Exam Flow

One full road test with 5 dynamic market windows

The exam prompt sends the agent into a simulated market. The agent must choose tools, submit an action, describe order intent, respect confirmation rules, and leave an auditable reason.

  1. License card
  2. Market window
  3. Agent answer
  4. Examiner verdict
